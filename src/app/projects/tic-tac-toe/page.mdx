A classic Tic-Tac-Toe game built as my first React project, based on the official React tutorial.

## What I learned

- React component structure and composition
- State management with `useState`
- Handling user interactions and events
- Conditional rendering
- Setting up a project with Vite and TypeScript

## Technical details

The game features a 3x3 grid where two players take turns placing X and O marks. It includes move history tracking and the ability to jump back to any previous move.
